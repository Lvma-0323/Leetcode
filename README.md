#### 双指针
指的是在遍历对象的过程中，不是普通的使用单个指针进行访问，而是使用两个**相同方向（快慢指针）** 或者**相反方向（对撞指针）** 的指针进行扫描，从而达到相应的目的。前者解决主要解决**链表**中的问题，比如典型的判定链表中是否包含环；后者主要解决**数组（或者字符串）** 中的问题，比如二分查找。
